Demo of Sidora Analysis API for Islandora

This is a demo of the Islandora module Sidora Analysis API, abbreviated sianis.

Analyses are performed by workflows, which run on a workflow server, through a module called Generic Workflow, abbreviated gflow. This module has its own demo interface here.

Workflows are constructed with the Taverna Workbench, and run by the Taverna Server, as configured at the Generic Workflow module block.

See about Taverna here.

The data for the analyses are selected from the Sidora repository by functionality in sianis.

In this demo you may choose to

  • select workflow first,
    • then select the data, where you may
      • select project first,
      • select geographic location first

or you may choose to

  • select data first,
    • then select the workflow
as seen below, where the selection of each data type shall be tailored in the user interface to assist the user in discovery and selection of the data.
